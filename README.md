-------------------
COMP3000 - Cinema booking site 
-------------------
Launch the application by extracting the files from the zip folder. 

Then in a terminal type 'node connect.js' - this connects the site to MongoDB
(shift + c to end the connection). 

the site is hosted on http://localhost:3000
